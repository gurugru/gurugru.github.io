Integer Factorization

1103 Integer Factorization (30分)

The K−P factorization of a positive integer N is to write N as the sum of the P-th power of K positive integers. You are supposed to write a program to find the K−P factorization of N for any positive integers N, K and P.

Input Specification:

Each input file contains one test case which gives in a line the three positive integers N (≤400), K (≤N) and P (1<P≤7). The numbers in a line are separated by a space.

Output Specification:

For each case, if the solution exists, output in the format:

where n[i] (i = 1, …, K) is the i-th factor. All the factors must be printed in non-increasing order.

Note: the solution may not be unique. For example, the 5-2 factorization of 169 has 9 solutions, such as 122+42+22+22+12, or 112+62+22+22+22, or more. You must output the one with the maximum sum of the factors. If there is a tie, the largest factor sequence must be chosen – sequence { a1,a2,⋯,a**K } is said to be larger than { b1,b2,⋯,b**K } if there exists 1≤L≤K such that a**i=b**i for i<L and a**L>b**L.

If there is no solution, simple output Impossible.

生词

factorization 因子分解

题目大意

对正整数N进行k-p因子分解，其中k为分解得到的因子个数，p为幂。

要求

所输出的因子序列的和在所有因子序列中最大，存在因子和相同的多个因子序列时输出字典序最大的序列，且因子序列为非递增形式。

测试点

测试点2 考察输出序列的字典序 栗子 82 18 1

测试点5 卡时间 剪枝 建造因子幂次结果表

思路

最先想到可以使用递归进行分治，k-p因子分解问题可以分化为 (k-1)-p因子分解问题，而参数N的值为上一层参数N减去这一层因子的p幂。设计递归函数 factor(int delta,int index ,int p),根据要求非递增要求，加入前一项因子选择pre进行剪枝，更新递归函数factor(int delta,int index ,int p ,int pre)。函数主体部分，采用for循环对因子选项进行遍历，设立基线条件为index==1，即选择最后一个因子时进行遍历返回。于是可以得到整个递归函数

第二种方法 DFS

首先问题的解决步骤为进行k次因子选择，设可选择因子个数为R,则该问题空间为k层满R叉树，下图所示R==2,K==3;采用DFS对该树进行遍历，从root出发到叶子结点返回。
