字符串操作函数传参入手了解C语言指针

栗子代码如下：

首先要讲的是字符串操作函数，如上述代码中ptr()进行字符串赋值操作，字符串将其首地址赋值给指针，存在指针本身值的改变，在函数中修改变量的值需要传入变量的地址，因此传入指针的地址&str，可以看到在output第二行和第六行看到指针本身值改变的情况。另外再讨论一下函数内部赋值中指针的情况，定义一个指针变量ch指向str，可以发现，其本身值与str相同，都为0060FEFC，而存储该指针变量的地址不同，分别为0060FECC 与0060FEF8，可以发现不同变量的存储地址不同。

于是，可以发现，地址与变量关联，每一个变量C都会给他开辟存储空间，假设指针p指向变量a, 则指针的地址存储变量a的地址即指针p的值，变量a的地址存储a的值。总是从地址中获取值，当地址中的值为其他变量的地址时，该变量则可以认为是指针（也可以是多重指针）。
